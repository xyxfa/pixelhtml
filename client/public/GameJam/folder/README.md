# 时间倒流 (Time Rewind) - 资源文件说明

## 需要的资源文件

请将以下资源文件放置在此文件夹中：

1. **bg-seamless.png** - 背景平铺图（用于滚动视差效果）
   - 建议尺寸：1920x1080 或更大
   - 格式：PNG（支持透明）
   - 风格：像素风格，与游戏主题一致

2. **标题.jpg** - 主展示图片（显示在游戏信息卡片中）
   - 建议尺寸：1200x800 或类似比例
   - 格式：JPG 或 PNG
   - 内容：游戏标题图或主视觉图

3. **游戏截图**（可选，用于画廊展示）
   - screenshot-1.png
   - screenshot-2.png
   - screenshot-3.png
   - 等等...

## 当前配置

- **主题色**：金色 (#fbbf24)
- **年份**：2024
- **类别**：GameJam
- **标签**：Godot, Puzzle, Time, GGJ

## 注意事项

- 所有图片路径都是相对于 `/GameJam/时间倒流/` 的
- 如果暂时没有资源，可以先使用占位图片，后续再替换
- 背景图建议使用可平铺的图案，以实现滚动视差效果
